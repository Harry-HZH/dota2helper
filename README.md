# dota2helper

### 1.项目介绍

本项目仅为本人练手，含有移动端展示部分和后台数据录入部分

#### 1.1环境

vue@2.6 nodejs@14 mongodb@4.4.3

### 2.安装说明

admin文件夹下 npm install

server文件夹下 npm install

web文件夹下 npm install

### 3.使用说明

admin文件夹下 npm run serve

server文件夹下 npm run serve

web文件夹下 npm run serve
